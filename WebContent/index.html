Technologies used



  • Rest-Jersey Implementation
  • JAXB-(for marshalling and unmarshalling json/xml types)
  • Spring Data JPA
  • Hibernate Implementation
  • MySql
  • Tomcat 7
  • Advanced Rest client (chrome extension)



To get started deploy the application in tomcat and hit "http://localhost:yourport/ZenDirectory/rest/query/insert" tables will be created automatically and inserts data. for more search options insert some other data in "public void insertPerson()" method from EmployeeDaoImpl class.

Open advance rest client and enter url http://localhost:9958/ZenDirectory/rest/query/name?name='saam' and add headers "accept":"application/json" or "application/xml" and the response will display in specified format below.

I implemented Dao in JPA specific so we can change ORM from Hibernate to any other ORM framework by adding jars and small configuration changes in spring-configuration.xml file.
